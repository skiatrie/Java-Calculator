# Java Calculator 

A command-line java calculator application exercising Java fundamentals including input validation, state management, and operator handling
---

# Concepts demonstrated

- **Scanner input handling** for numeric and operator input
- **State management** using instance variables for: 
  - current display values
  - stored operands
  - pending operation
- **Conditional logic** for clean operator usage
- **Edge case handling** for division by 0, trailing decimal points, or consecutive operators
- **String formatting** for clean numeric output (integer vs. decimal)

# How to Run

```bash
javac App.javac
java App.java
```
Supported operations: `+`, `-`, `*`, `/`, `sqrt`. Press `=` to see the result.
